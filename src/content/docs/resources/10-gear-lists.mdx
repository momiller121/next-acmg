Whether you are planning a day trip or an expedition these activity specific gear lists will help you remember and organize those crucial items.


## Hiking

**Clothing**<br/>
( ) Wool and/or Synthetic Socks<br/>
( ) Base Layer Top - Synthetic or wool<br/>
( ) Base Layer Bottoms - Synthetic or wool - optional for cool or winter days<br/>
( ) Mid-Insulation Layer Top - Pile (fleece), light synthetic fill or wool sweater/jacket<br/>
( ) Hiking Pants- Synthetic multipurpose pants<br/>
( ) Light Wind Shell - Windproof, water resistant layer for high aerobic work<br/>
( ) Weather Layer Top - Gore-Tex or other waterproof breathable jacket, or coated nylon<br/>
( ) Weather Layer Pants - Gore-Tex or other waterproof breathable pants, or coated nylon<br/>
( ) Warm Outer Jacket - Down or synthetic for cool or winter days<br/>
( ) Toque - Wool or Synthetic<br/>
( ) Brimmed Cap - Preferably wool or synthetic but cotton will do if other unavailable<br/>
( ) Face Warmer - Scarf, neck tube, face mask - optional, for winter trips<br/>
( ) Light Gloves - Wool, synthetic or leather<br/>
( ) Warm Gloves or Mitts - for cool or winter days<br/>
( ) Gaiters - for wet weather<br/>
( ) Handkerchief<br/>
**Warm Weather Hiking**<br/>
( ) Shorts - or convertible pants<br/>
( ) Sun Hat<br/>
( ) Cotton T-shirt<br/>
**Personal Equipment**<br/>
( ) Pack<br/>
( ) Sunglasses<br/>
( ) Sun and Lip Cream<br/>
( ) Head Lamp<br/>
( ) Water Bottle or Hydration System<br/>
( ) Lunch<br/>
( ) Blister Kit<br/>
( ) Pocket Knife<br/>
( ) Camera - Optional<br/>
( ) Binoculars - Optional<br/>
( ) Walking stick or Ski Pole<br/>
( ) Umbrella<br/>
( ) Hiking Boots<br/>
**Group Equipment**<br/>
( ) Altimeter<br/>
( ) Map and Compass<br/>
( ) Route Book/Description<br/>
( ) Natural History References etc. - optional<br/>
( ) First Aid Kit<br/>
( ) Emergency Sac - Nylon tarp or envelope sack to cover group or injured member<br/>
( ) Group Repair Kit - May substitute for personal repair kits or supplement them<br/>
( ) Two-way Radio or Cell Phone - Check for coverage and frequencies in your area<br/>
( ) GPS - Optional depending on trip<br/>
( ) Small Stove and Pot - Optional depending on trip<br/>
( ) Water Filter - optional on short trips where you can carry required water<br/>
